# Configuration Reference

Ward DND AI is configured through two mechanisms:
1. **`.env` files** — secrets and environment-specific values (never committed)
2. **`settings.json`** — app preferences editable from the Settings tab

---

## Environment Variables (`.env`)

Copy `.env.example` to `.env` and fill in your values.

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `OPENAI_API_KEY` | Yes | — | Your OpenAI API key |
| `APP_ENV` | No | `development` | Active environment: `development`, `production`, `test` |
| `VAULT_PATH` | No | `./Obsidian` | Override the vault path from settings.json |
| `COMPLETION_MODEL` | No | `gpt-4o` | OpenAI chat model |
| `EMBEDDING_MODEL` | No | `text-embedding-3-small` | OpenAI embedding model |
| `LOG_LEVEL` | No | `INFO` | Logging verbosity: `DEBUG`, `INFO`, `WARNING`, `ERROR` |
| `MAX_TOKENS` | No | `4000` | Max tokens per AI response |

Environment variables always take precedence over `settings.json` values.

### Environment files load order

1. `.env` — base secrets (always loaded)
2. `.env.{APP_ENV}` — environment overrides (loaded if file exists)

## settings.json

Located at `Ward_DND_AI/config/settings.json`. Editable from the Settings tab or directly.

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `VAULT_PATH` | string | `./Obsidian` | Path to your Obsidian vault |
| `VAULT_TYPE` | string | `hybrid` | Storage backend type (`hybrid`) |
| `APP_NAME` | string | `Ward DND AI` | Application window title |
| `THEME` | string | `Light` | UI theme (`Light` or `Dark`) |
| `FONT_SIZE` | string | `Medium` | UI font size (`Small`, `Medium`, `Large`) |
| `STARTUP_TAB` | string | `Dashboard` | Tab shown on launch |
| `SHOW_TOOLTIPS` | bool | `true` | Enable hover tooltips |
| `COMPACT_MODE` | bool | `false` | Dense UI layout |
| `COMPLETION_MODEL` | string | `gpt-4o` | OpenAI chat completion model |
| `EMBEDDING_MODEL` | string | `text-embedding-3-small` | OpenAI embedding model |
| `MAX_TOKENS` | int | `4000` | Max tokens per AI response |
| `LOG_FILE` | string | `logs/app.log` | App log file path |
| `LOG_LEVEL` | string | `INFO` | Log verbosity |
| `AUTO_REFRESH_INTERVAL` | int | `300` | Vault refresh interval in seconds |
| `AI_BACKENDS` | dict | see below | Maps AI tasks to backend plugins |

### AI_BACKENDS default

```json
{
  "ask": "openai",
  "summarize": "openai",
  "suggest_tags": "openai",
  "propose_links": "openai",
  "search_context": "loreai"
}
```

## Logs

| File | Description |
|------|-------------|
| `logs/app.log` | General application log (rotating, 1MB max, 3 backups) |
| `logs/audit.log` | CRUD audit trail — every create/update/delete |
| `logs/crash_*.log` | Crash reports generated by the crash handler |
| `logs/last_crash_summary.txt` | Summary of the most recent crash |
| `Ward_DND_AI/ai/ai_usage_log.csv` | OpenAI API usage and cost tracking |
