# Version Release Checklist

- [ ] Code is fully tested and works
- [ ] All TODOs resolved or documented
- [ ] `requirements.txt` is up to date
- [ ] Version/tag updated if needed
- [ ] Commit: `git commit -am "release: vX.Y.Z"`
- [ ] Tag version: `git tag vX.Y.Z`
- [ ] Push everything: `git push && git push --tags`
- [ ] Backup/export any vault data
